# 유형별 참고자료 정리: 박시훈

## 1. 제출 목적

과제 조건에서 요구한 “유형별 참고 자료”를 별도 파일로 정리한 자료이다. 본 연구 주제는 공공데이터와 설명가능한 AI를 활용한 광주광역시 소상공인 입지 분석 시스템 설계이다.

## 3. 자료별 정리

### 3.1 단행본

**김영갑, 상권분석전략연구회. 2024. 『소상공인을 위한 빅데이터 상권분석』. 4판. 서울: 이프레스.**

이 자료는 소상공인 상권 분석에서 활용할 수 있는 빅데이터 기반 분석 방법을 설명하는 단행본이다. 본 연구에서는 상권 분석의 핵심 변수로 유동인구, 배후인구, 업종 밀집도, 임대료 수준 등을 설정하는 근거로 활용할 수 있다.

활용 위치:

- 서론: 데이터 기반 입지 분석 필요성
- 본론: 입지 분석 변수 설계
- 본론: 수요, 경쟁, 접근성, 비용 부담 proxy 설명

인용 방식:

```text
김영갑과 상권분석전략연구회는 상권 분석에서 유동인구, 배후인구, 업종 밀집도, 임대료 수준 등을 종합적으로 검토해야 한다고 설명한다.
```

주의:

직접 인용을 넣으려면 책의 실제 페이지를 확인해야 한다. 페이지 확인 전에는 간접 인용으로 쓰는 것이 안전하다.

### 3.2 정부 통계·보고서

**중소벤처기업부, 소상공인시장진흥공단. 2026. 『2024년 기준 소상공인실태조사 결과』. 세종: 중소벤처기업부.**

이 자료는 2024년 기준 소상공인 규모, 종사자 수, 경영 애로사항, 창업비용 등을 제시하는 공식 통계 자료이다. 본 연구에서는 소상공인이 지역경제에서 차지하는 비중과 데이터 기반 의사결정 지원의 필요성을 제시하는 데 활용한다.

활용 위치:

- 서론: 연구 필요성
- 본론: 소상공인의 경제적 비중
- 결론: 정책적 활용 가능성

직접 인용 후보:

```text
“소상공인 기업체는 613만 개”
```

간접 인용 후보:

```text
중소벤처기업부와 소상공인시장진흥공단의 조사에 따르면, 소상공인은 기업체 수와 종사자 수 측면에서 지역경제의 중요한 비중을 차지한다.
```

### 3.3 학술논문 1

**장호준, 이기환, 이희상. 2024. 「공공데이터와 설명가능한 AI 기법을 활용한 자영업 상권 분석과 장기 생존 예측」. 『대한산업공학회지』 50(3): 173-188.**

이 논문은 공공데이터와 설명가능한 AI를 활용해 자영업 상권 분석을 수행한 선행연구이다. 본 연구에서는 공공데이터, 머신러닝, XAI를 결합한 상권 분석 시스템 설계의 이론적 근거로 활용한다.

활용 위치:

- 본론: 선행연구 검토
- 본론: XAI 기반 설명 구조 필요성
- 결론: 본 연구의 차별성 설명

직접 인용 후보:

```text
“중요한 역할”
```

간접 인용 후보:

```text
장호준·이기환·이희상의 연구는 공공데이터와 설명가능한 AI가 자영업 상권 분석에 활용될 수 있음을 보여준다.
```

본 연구와의 연결:

```text
해당 연구가 서울 등 기존 사례 중심이라면, 본 연구는 광주광역시를 대상으로 한 지역 맞춤형 시스템 설계에 초점을 둔다.
```

### 3.4 학술논문 2

**김철민. 2025. 「지역기반의 소상공인 매출액 예측모형 개발」. 『한국지역개발학회지』 37(2): 123-146.**

이 논문은 지역 단위 상권 데이터를 활용한 머신러닝 기반 분석 사례이다. 본 연구에서는 지역 기반 상권 데이터 분석이 가능하다는 선행 근거로 활용하되, 본 연구의 구현 결과는 직접적인 재무 결과 산출이 아니라 입지 적합도와 영업 유지 proxy 중심이라는 차이를 명확히 한다.

활용 위치:

- 본론: 머신러닝 기반 상권 분석 선행연구
- 본론: 지역 단위 분석의 필요성
- 결론: 향후 데이터 확장 방향

직접 인용 후보:

```text
“XGBoost 머신러닝 알고리즘”
```

간접 인용 후보:

```text
김철민의 연구는 지역 단위 상권 자료와 머신러닝 기법의 결합 가능성을 보여준다.
```

주의:

본 논문 초고에서는 해당 연구의 목적을 소개하되, 본 연구가 동일한 산출물을 제공한다고 쓰지 않는다. 본 연구는 입지 비교와 후보지 필터링에 초점을 둔다.

### 3.5 인터넷 자료

**소상공인시장진흥공단. n.d. 「소상공인365 상권분석 서비스」. https://bigdata.sbiz.or.kr/.**

이 자료는 실제 공공 상권 분석 서비스 사례이다. 본 연구에서는 기존 서비스가 어떤 정보를 제공하는지 설명하고, BizSpot AI가 후보지 점수와 설명을 함께 제공한다는 점을 비교하는 데 활용한다.

활용 위치:

- 서론: 기존 서비스 현황
- 본론: 기존 서비스와 제안 시스템 비교
- 결론: 의사결정 지원 도구로서의 확장 가능성

직접 인용 후보:

```text
“업종 분포, 유동인구”
```

간접 인용 후보:

```text
소상공인365는 상권 단위의 업종, 인구, 소비 관련 자료를 제공하지만, 사용자가 여러 지표를 직접 해석해야 한다는 점에서 설명형 시스템의 필요성이 남아 있다.
```

## 6. 논문 작성 기준

본 논문은 시카고 양식의 각주-참고문헌 규정에 따라 작성한다. 본문에서 자료를 직접 인용할 경우 따옴표를 사용하고, 각주에 출처를 밝힌다. 간접 인용의 경우 원문의 핵심 내용을 연구자의 문장으로 바꾸어 서술하되, 해당 문장 끝에 각주를 달아 출처를 제시한다. 참고문헌은 글 마지막에 별도로 정리한다.
